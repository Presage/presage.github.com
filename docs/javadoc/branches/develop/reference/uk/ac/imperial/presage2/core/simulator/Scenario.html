ScenarioBuilder

This is a Scenario which allows you to build up your scenario step by step before submitting it to the Simulator. 

Class Overview

A Scenario describes the runtime components of a simulation which the simulator must interact with. These comprise of the Participants of the system, the other TimeDriven elements, and the Plugins. The Scenario will likely have to construct other elements which the above require such as network components and the environment.
