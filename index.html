Presage

A tool for rapid prototyping of agent societes.

About

PRESAGE is a simulation platform for rapid prototyping of Agent Societies. It enables designers to investigate the effect of agent design, network properties and the physical environment on individual agent behaviour and long-term collective global performance.

Originally written by Brendan Neville it is currently being developed by Sam Macbeth as part of his PhD at Imperial College London.

Publications

  • Jeremy Pitt, Brendan Neville, Sam Macbeth, Hugo Carr, "Animation of Open Multi-Agent Systems" Agent-Directed Simulation 2011
  • Brendan Neville and Jeremy Pitt, "PRESAGE: A Programming Environment for the Simulation of Agent Societies" Proceedings AAMAS Workshop on Programming Multi-agent Systems (ProMAS) 2008
